AMS v3.0 사용 가이드

하카코리아 자산 관리 시스템 (Asset Management System) 전 기능 사용 설명서

⊞시스템 개요
📌 AMS란?

하카코리아 직영점에 물류 자산(소모품, 비품, 유니폼 등)을 출고·수거하고 재고를 관리하는 웹 기반 시스템입니다.

단일 HTML 파일로 구성되어 있으며 Supabase(클라우드 DB)를 백엔드로 사용합니다. 인터넷이 연결된 환경에서 파일을 열거나, GitHub Pages 주소에 접속하면 사용 가능합니다.

🗂 DB 테이블 구성

inventory · requests · shipments · pickups · logistics · store_assets · bag_shipping · uniform_requests · uniform_pickups · uniform_manager_stock

🔔 텔레그램 알림

4가지 이벤트에서 알림 발송:

📦 자산 신청 / 🔄 수거 신청 / 📋 구매요청서 제출 / 🛍 쇼핑백 출고 계산

📥 전체 DB 내보내기

대시보드 우상단 "📥 전체 DB 엑셀 내보내기" 버튼으로 전체 9개 테이블을 xlsx로 다운로드합니다.

⚠️ 시스템 사용 전 인터넷 연결 상태를 확인하세요. DB는 클라우드(Supabase)에 저장되므로 오프라인에서는 데이터 조회·저장이 불가합니다.

⊞대시보드
전체 현황 요약 화면
사이드바 → 대시보드
  • 4개 요약 카드: 물류재고 품목 수 / 신청 대기 건수 / 이번달 출고 건수 / 수거 대기 건수
  • 물류재고 부족 알림 테이블 — 재고 10개 이하 품목을 우선 표시
  • 최근 신청 현황 테이블 — 최근 출고 신청 목록 미리보기
  • 우상단 "📥 전체 DB 엑셀 내보내기" 버튼으로 전체 데이터 xlsx 다운로드
▦물류 재고
창고 재고 현황 카드
사이드바 → 물류 재고
  • 등록된 모든 품목의 재고를 카드 형태로 표시
  • 재고 상태에 따라 색상 구분: 재고없음 부족(10↓) 정상
  • 필터 버튼으로 재고없음 / 부족 / 고정출고 / 정상 / 🔧고장있음 / 📦반납재고 분류 조회
  • 카테고리 필터: 소모품 / 비품 / 인쇄물
  • 카드 클릭 → 수량 직접 수정, 입출고 이력 확인 가능
  • 🔧 고장 재고: 고장 수거 처리 시 별도 고장 재고 수량 증가
  • 📦 반납 재고: 유니폼 수거 입고확인 시 별도 반납재고 수량 증가
💡 재고 카드에서 직접 수량을 수정하면 즉시 DB에 반영됩니다. 카드의 초록색 테두리 = 고정 출고 품목(정기 출고 설정된 품목)입니다.
🏪매장 관리
직영점 정보 등록 및 수정
사이드바 → 매장 관리
  • 신규 매장 등록: 매장코드 / 매장명 / 배송코스 / 주소 입력 후 "+ 등록"
  • 등록된 전체 매장 목록 조회 (매장명·코드·코스 검색 가능)
  • 각 행 우측 수정 / 삭제 버튼으로 정보 변경
  • 여기에 등록된 매장 정보는 출고 신청·수거 신청에서 자동완성으로 연동됩니다
⚠️ 매장명은 출고 신청·수거 신청·유니폼 신청 등 전 기능에서 공통으로 사용됩니다. 매장명을 변경할 경우 기존 이력에는 이전 이름이 그대로 유지됩니다.
◫자산 현황 (출고 관리)
물류 재고 단위 전체 현황 테이블
사이드바 → 출고 관리 → 자산 현황
  • 자산코드 / 자산명 / 카테고리 / 현재재고 / 총보유 / 출고누계 / 고정출고 / 비고 컬럼
  • + 자산 추가: 신규 품목 등록 (코드·이름·카테고리·수량 입력)
  • ⬇ 엑셀 다운로드: 전체 자산 현황을 xlsx로 내보내기
  • 검색창으로 품목명·코드 검색 가능
  • 우측 수정 아이콘으로 개별 자산 수량·정보 수정 가능
↗출고 신청
자산 출고 신청 등록 및 목록
사이드바 → 출고 관리 → 출고 신청
  • 상단 신청 폼 (접기/펼치기 가능): 신청일자, 담당자, 소속팀, 보낼 매장명, 배송코스, 매장주소, 비고, 신청 품목
  • 매장명 입력 시 배송코스·주소 자동 입력
  • 품목을 여러 개 추가 가능 (+ 품목 추가 버튼)
  • 신청 목록: 신청일 + 신청자 + 매장 기준으로 카드 그룹화, 더블클릭으로 접기/펼치기
  • 상태 필터: 대기중 승인 출고완료 반려
  • 대기중 상태 행에만 삭제 버튼 표시
  • 카드 우상단 📥 엑셀 버튼으로 해당 신청 건 엑셀 다운로드

⊞일괄 출고 신청
엑셀 파일로 대량 출고 신청
사이드바 → 출고 관리 → 일괄 출고 신청
  • ⬇ 양식 다운로드 후 엑셀에 여러 매장·품목 입력
  • 작성 완료된 엑셀 파일을 업로드하면 미리보기에서 오류 검증
  • 오류 없으면 한 번에 다수 신청 등록 (각 행이 개별 신청으로 등록됨)
  • 등록 후 일반 출고 신청과 동일한 흐름으로 처리
⬡출고 관리
출고 대기 목록 및 처리
사이드바 → 출고 관리 → 출고 관리
  • 출고 대기 목록: 승인 대기 중인 신청 건 목록 (카드 그룹 형태)
  • 각 카드에서 승인 → 출고 처리 버튼으로 순차 처리
  • 출고 처리 시 물류 재고 자동 차감, 출고 날짜·담당자 기록
  • 출고 완료 기록: 처리 완료된 건 하단 테이블에 이력 표시
💡 출고 처리 완료 시 물류 재고에서 해당 수량이 자동으로 차감됩니다. 반드시 재고 수량을 확인하고 처리하세요.
◎이력 조회
전체 출고 이력 검색
사이드바 → 출고 관리 → 이력 조회
  • 모든 출고 신청 이력을 카드 그룹 형태로 조회
  • 검색창으로 매장명·품목명·담당자 검색 가능
  • 각 카드에서 📥 엑셀 다운로드 가능

사이드바 → 수거 관리 → 수거 신청
  • 상단 수거 신청 폼 (접기/펼치기 가능): 요청일자, 담당자, 소속팀, 수거 사유, 비고, 수거 매장명, 배송코스, 매장주소, 수거 품목
  • 수거 사유: 📦 일반수거 / 🔧 고장 / 📋 운영진 지시사항 / 📝 기타
  • 매장명 입력 시 배송코스·주소 자동 입력
  • 수거 신청 목록: 상태별 필터 탭 — 대기중 진행중 수거완료 반려
  • 비고 내용이 수거 관리 탭과 수거 이력 탭에도 표시됨

✦수거 관리
수거 확인 및 재고 자동 반영
사이드바 → 수거 관리 → 수거 관리
  • 수거 대기 목록: 아직 수거 확인되지 않은 건 목록
  • "수거 확인" 버튼 클릭 → 수거일자·확인 담당자 입력 → 확인 처리
  • 수거 확인 완료 시 물류 재고 자동 증가
  • 수거 완료 기록: 처리 완료 이력 하단 테이블에 표시
◐수거 이력
전체 수거 이력 조회
사이드바 → 수거 관리 → 수거 이력
  • 모든 수거 신청 이력 (진행중/완료/반려 포함) 전체 테이블 조회
  • 컬럼: 수거번호 / 요청일자 / 수거일자 / 수거확인일자 / 신청자 / 품목명 / 수량 / 배송코스 / 수거 매장 / 주소 / 비고 / 배송비고 / 상태
  • 검색창으로 매장명·품목명·담당자 검색 가능
🏢자산 관리 (본사/직영점 IT 자산)
하카코리아 IT 자산 관리 시스템
사이드바 → 자산 관리 → 본사/직영점 자산현황, 자산·직원 등록, 스티커 출력
  • AMS 내부에 별도 iframe으로 탑재된 독립 시스템
  • 데이터는 브라우저 localStorage에 저장 (클라우드 DB 미사용)
🏢 본사 자산현황
  • 본사 직원(부서별) IT 자산 목록 조회
  • 검색·필터·수정·삭제 가능
  • 품목 배지 표시 (MO=모니터, PC=데스크탑, NO=노트북 등)
🏪 직영점 자산현황
  • 직영점별 IT 자산 목록 조회
  • 검색·필터·수정·삭제 가능
  • 875개 이상의 직영점 자산 관리
➕ 자산/직원 등록
  • 조직도/부서 연관검색 → 사원명 연관검색
  • 품목분류 선택 시 기존 모델명 자동완성
  • 취득일 입력 + 채번 자동 생성 (예: QU-MO-001, 21051-PC-150)
🏷 스티커 출력
  • 63.5×38.1mm 21칸 A4 라벨지 출력
  • 이름/직영점명 검색 + 체크박스 선택
  • 부서/직영점별 일괄 선택 모달 (전체선택 포함)
  • 스티커 내용: 관리번호 / 소속팀 / 품목명 / 모델명 / 취득일 / 사용자
📋구매요청서
물품 구매 요청 등록 및 이력
사이드바 → 주문 관리 → 구매요청서
  • 직접 입력: 담당자·구매요청일 입력 후 품목 행 추가 (품목명·요청수량·비고)
  • ⬇ 양식 다운로드 → 엑셀로 작성 후 ⬆ 엑셀 등록 업로드도 가능
  • "📤 구매요청 제출" 클릭 시 DB 저장 + 텔레그램 알림 발송
  • 구매요청 이력: 요청번호·날짜·담당자·품목수·상태 테이블 조회
📦비품주문표
주문 승인 및 입고 확인
사이드바 → 주문 관리 → 비품주문표
  • 구매요청서에서 제출된 요청 건을 주문관리팀이 승인하는 화면
  • 검색·날짜범위·상태 필터로 특정 건 조회
  • 물류팀에서 입고 확인 처리 → 재고 자동 반영
📥쇼핑백/무지봉투 — 데이터 업로드
포스 재고파일 업로드 및 출고량 계산
사이드바 → 쇼핑백/무지봉투 출고 → 데이터 업로드
  • 플레이엠디 재고파일 (서울/경기/인천/제주 매장): 매장재고현황 xlsx 업로드
  • 돌핀포스 재고파일 (부산/대구/울산/경남 매장): 수불_집계_현황 xlsx 업로드
  • 드래그&드롭 또는 클릭 업로드 모두 지원
  • 업로드 후 미리보기 테이블: 포스구분 / 매장명 / 매장코드 / 쇼핑백 현재수량 / 무지봉투 현재수량
  • "▶ 출고량 계산 & 저장" 클릭 시 출고 규칙에 따라 자동 계산 후 DB 저장

📦쇼핑백/무지봉투 — 출고 결과
출고 결과 확인 및 엑셀 다운로드
사이드바 → 쇼핑백/무지봉투 출고 → 출고 결과
  • 출고 합계 요약: 현재 미출고 건 기준 쇼핑백·무지봉투 총 묶음 수 표시
  • 업로드 배치별로 카드 구성, 각 카드에 쇼핑백 소계 / 무지봉투 소계 표시
  • 출고 처리 전: 출고필요 배지 + 출고 예정 수량 표시
  • 출고 완료 후: 출고완료 배지 + 실제 출고된 수량 기록
  • 📥 엑셀 버튼: 포스사별 엑셀 파일 다운로드 (플레이엠디 / 돌핀포스 별도)
  • 🚚 일괄 출고 버튼: 해당 배치 전체를 한번에 출고 처리
  • 🗑 배치 삭제: 해당 배치 전체 데이터 삭제
  • 매장명 검색으로 특정 매장만 필터링 가능
⚠️ 출고 완료 처리 후에는 해당 배치가 "출고완료" 상태로 변경되며, 합계 카운트에서 제외됩니다. 출고완료 기록은 하단에 이력 형태로 유지됩니다.

👕유니폼 신청
유니폼 출고 신청 및 목록
사이드바 → 유니폼 관리 → 유니폼 신청
  • 개별 신청 폼 (접기/펼치기 가능): 작성일자·신청자·직영점명·수령 매니저 성함·구분(택배/직납)·신청 품목
  • 직영점명·수령 매니저 자동완성 지원
  • 품목 행: 유니폼 품목명·사이즈·수량·사유 입력
📥 엑셀 일괄 등록
  • ⬇ 양식 다운로드: 3개 시트 포함 — 📋 입력양식 / 🏪 매장목록 / 👕 품목목록
  • 업로드 즉시 매장명·품목명 DB 대조 검증, 오류 행 빨간색 강조
  • 오류 없으면 "일괄 등록" 클릭 → batch_id로 묶음 등록
  • 일괄 등록된 항목은 보라색 "일괄등록 N건" 배지로 구분
유니폼 신청 목록 (flat 테이블)
  • 컬럼: 작성일자 / 신청자 / 직영점 / 수령매니저 / 구분 / 유니폼 / 사이즈 / 수량 / 사유 / 비고 / 상태 / 출고일자 / 출고비고 / 액션 / 삭제
  • 대기중 상태 행에만 출고하기 버튼 표시
  • 출고 처리 시 물류재고 차감 + 관리대장 자동 업데이트

↩유니폼 수거
유니폼 반납 수거 신청 및 확인
사이드바 → 유니폼 관리 → 유니폼 수거
  • 수거 신청 폼: 작성일자·신청자·직영점명·반환 매니저 성함·구분(택배/직납)·수거 품목
  • 수거 목록 테이블: 번호/작성일자/신청자/직영점/반환매니저/구분/유니폼/사이즈/수량/사유/비고/상태/수거예상일/수거일자/수거비고/액션/삭제
  • "수거예상일 입력" 버튼으로 예상 수거일 등록 (매장 안내용)
  • "수거확인" 버튼 클릭 → 수거일자 입력 → 확인 처리
  • 수거 확인 완료 시 물류 반납재고 증가 + 관리대장 수량 감소

📋수불 이력
유니폼 출고·수거 통합 이력
사이드바 → 유니폼 관리 → 수불 이력
  • 유니폼 신청(출고)과 수거 이력을 한 화면에서 통합 조회
  • 탭 필터: 전체 / 신청(출고) / 수거
  • 컬럼: 구분/작성일자/출고(수거)일자/도착일자/신청자/직영점/매니저/유니폼/사이즈/수량/사유/비고(신청)/비고(처리)
  • 검색창으로 매장명·매니저명·품목명 검색 가능
📊관리대장
직영점 매니저별 유니폼 보유 현황
사이드바 → 유니폼 관리 → 관리대장
  • 직영점별 매니저 목록과 하계·춘추·동계 유니폼 보유 수량 표시
  • 유니폼 출고 시 수량 자동 증가, 수거 확인 시 수량 자동 감소
  • 필터: 전체 / 0개 있음 / 하계 0개 / 춘추 0개 / 동계 0개
  • 매니저명으로 검색 가능
  • 📥 엑셀 내보내기 버튼으로 전체 관리대장 xlsx 다운로드
  • 각 매니저 행에서 수정 버튼으로 재직상태·사이즈·수량 직접 수정 가능
💡 관리대장의 수량은 유니폼 신청·수거 처리 시 자동으로 업데이트됩니다. 수동 수정이 필요한 경우에만 수정 버튼을 사용하세요.
